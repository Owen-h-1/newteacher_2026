## 🚀 如何使用

### 步骤 1: 访问系统

打开浏览器，访问以下地址：
- **主界面**: http://localhost
- **AI 数字人页面**: http://localhost/chatvrm/

### 步骤 2: 找到人脸识别按钮

在 AI 数字人页面的**右上角**，您会看到一个按钮：

```
┌─────────────────────────────┐
│                    [👤 表情] │  ← 这个按钮
│                             │
│      AI 数字人界面           │
│                             │
└─────────────────────────────┘
```

**按钮说明**：
- 未激活时显示：**👤 表情**（灰色背景）
- 激活后显示：**🎭 表情**（紫色背景）

### 步骤 3: 启动人脸识别

1. **点击按钮**：点击右上角的"表情"按钮
2. **展开面板**：右侧会滑入人脸识别面板
3. **允许摄像头**：浏览器会请求摄像头权限，点击"允许"
4. **开始识别**：系统会自动开始实时表情识别

### 步骤 4: 查看识别结果

面板会实时显示：

```

│  │                        │  │
│  │    [摄像头画面]          │  │
│  │                        │  │
│  └────────────────────────┘  │
│                              │
```

## 🎯 功能特性

### 1. 实时表情识别

系统可以识别 **8 种表情**：
- 😊 **开心** (happy)
- 😢 **悲伤** (sad)
- 😠 **愤怒** (angry)
- 😮 **惊讶** (surprised)
- 😐 **中性** (neutral)
- 🤔 **困惑** (confused)
- 😴 **无聊** (bored)
- 🎯 **专注** (focused)

### 2. 学习状态分析

系统会将表情映射为 **6 种学习状态**：
- 🟢 **专注学习** (engaged)
- 🟡 **好奇探索** (curious)
- 🔵 **中性状态** (neutral)
- 🟠 **困惑不解** (confused)
- 🔴 **沮丧挫败** (frustrated)
- ⚫ **分心走神** (disengaged)

### 3. 智能教学调整

当系统检测到需要调整教学时，会在页面顶部显示提示：

```
┌──────────────────────────────────────────────┐
│ 🎯 教学建议: 放慢节奏，详细解释当前知识点     │
│    原因: 检测到学生表现出困惑表情              │
│    优先级: 高 🔴                              │
└──────────────────────────────────────────────┘
```

### 4. AI 数字人情感反馈

AI 数字人会根据您的表情做出相应的反应：
- **困惑** → 数字人做出思考表情
- **沮丧** → 数字人做出悲伤表情
- **分心** → 数字人做出中性表情

## 🔧 技术架构

### 服务组成

```
浏览器 (前端)
    ↓
Nginx Gateway (http://localhost)
    ↓
├─ ChatVRM (前端界面) - 包含人脸识别按钮和面板
├─ ChatBot (AI 后端) - Ollama LLM
└─ FaceRecognition (人脸识别后端) - OpenCV
```

### API 端点

- **创建会话**: `POST /api/facerecognition/session/start`
- **分析表情**: `POST /api/facerecognition/analyze`
- **结束会话**: `POST /api/facerecognition/session/end`
- **会话状态**: `GET /api/facerecognition/session/{id}/status`

## 💡 使用建议

### 最佳实践

1. **光线充足**：确保面部光照均匀充足
2. **正对摄像头**：距离 50-100cm，正面朝向摄像头
3. **保持稳定**：避免快速移动或晃动
4. **清晰画面**：确保摄像头画面清晰

### 性能优化

- 识别间隔：300ms（每秒约 3 次）
- 历史窗口：保留最近 10 次分析结果
- 干预冷却：10 秒内不重复触发相同干预

## 🐛 故障排查

### 问题 1: 看不到"表情"按钮

**解决方案**：
1. 强制刷新浏览器：`Ctrl + F5` (Windows) 或 `Cmd + Shift + R` (Mac)
2. 清除浏览器缓存
3. 确认访问的是 `http://localhost` 而不是 IP 地址

### 问题 2: 摄像头无法打开

**解决方案**：
1. 检查浏览器摄像头权限设置
2. 确保没有其他应用占用摄像头
3. 尝试使用 Chrome 或 Edge 浏览器

### 问题 3: 识别不准确

**解决方案**：
1. 改善光线条件
2. 调整摄像头角度和距离
3. 确保面部完整出现在画面中

### 问题 4: 服务连接失败

**解决方案**：
```bash
# 检查服务状态
docker-compose -f d:\hu\docker-compose.integrated.yml ps

# 重启服务
docker-compose -f d:\hu\docker-compose.integrated.yml restart

# 查看日志
docker logs virtualteacher-chatvrm
docker logs facerecognition-backend
```
